# cinematica-de-fluidos
Este repositorio presenta un resumen detallado del Capítulo 4 del libro Mecánica de Fluidos: Fundamentos y Aplicaciones (4.ª edición) de Yunus A. Çengel. El objetivo es facilitar el aprendizaje mediante una síntesis de los conceptos clave, complementada con un set de flashcards diseñadas para reforzar la memorización.
